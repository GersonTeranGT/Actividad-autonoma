Modelo de Objetos de Documento (DOM)

¿Qué es el DOM?

El DOM (Document Object Model) es una representación estructurada de un documento HTML o XML. Permite a los lenguajes de programación, como JavaScript, acceder y modificar el contenido, la estructura y el estilo de una página web. El DOM organiza los elementos como nodos en forma de árbol, donde cada etiqueta HTML es un nodo que puede tener padres, hijos y hermanos(Escarcena, 2021).

Estructura del DOM

"La estructura del DOM se puede representar como un árbol en la que cada rama del árbol termina en un nodo, y cada nodo tiene objetos" (INESEM Business School, s.f.).

"Una propiedad importante de los modelos de estructura DOM es el isomorfismo estructural, es decir, si se utilizan dos implementaciones de DOM cualesquiera para crear una representación del mismo documento, crearán el mismo modelo de estructura, precisamente con los mismos objetos y relaciones" (INESEM Business School, s.f.).

Caracteristícas principales del DOM

Propiedades del objeto document

  • document.body: Devuelve el elemento <body>.
  • document.title: Devuelve o establece el título de la página.
  • document.imagenes: Devuelve una colección de todas las imágenes.
  • document.bgColor: Permite definir el color de fondo.

Otros objetos

  • Window: El objeto window es un objeto del navegador que siempre está en la parte superior de la jerarquía. Es creado automáticamente por el navegador.
  • Document: El objeto document tiene varias propiedades que hacen referencia a otros objetos que permiten acceder y modificar el contenido de la página web.
  • Form: Está representado por etiquetas de formulario.
  • Link: Está representado por etiquetas de enlace.
  • Anchor: Está representado por etiquetas href.

Métodos comunes

  • getElementById(id): Selecciona un elemento por su ID.
  • getElementsByTagName(tag): Devuelve una colección de elementos por etiqueta.
  • getElementsByName(name): Devuelve elementos por su atributo name.
  • write() y writeln(): Escriben contenido HTML directamente.

Navegación entre nodos

  • parentNode: Nodo padre.
  • childNodes: Lista de nodos hijos.
  • firstChild y lastChild: Primer y último hijo.
  • previousSibling y nextSibling: Hermanos anterior y siguiente.

Manipulación de contenido

  • innerHTML: Cambia el contenido HTML de un elemento.
  • setAttribute(): Cambia el valor del atributo del elemento HTML.

Tambien se puede cambiar el atributo o el estilo de un elemento utilizando:

  • elemento.atributo = valor;
  • elemento.style.propiedad = estilo;

Manejo de eventos

También se puede añadir manejadores de eventos mediante: 

document.getElementById(id).evento = function()

Otros datos

HTML

HTML se utiliza para estructurar las páginas web.

JavaScript

Javascript se utiliza para agregar comportamiento y dinamismo a las paginas web.

DOM

Javascript interpreta DOM fácilmente, es decir, no puede entender las etiquetas, por ejemplo, h1 en un documento HTML pero puede entender el objeto h1 en DOM y acceder utilizando diferentes funciones o métodos.

¿Para que sirve el DOM en desarrollo web?

El DOM es esencial para crear paginas web dinámicas e interactivas. Permite:

  • Modificar el contenido de la página sin recargarla.
  • Responder a eventos del usuario con clics, teclas o movimientos del mouse.
  • Actualizar estilos y atributos de elementos en tiempo real.
  • Crear, eliminar o mover elementos dentro del documento.

Gracias al DOM, JavaScript puede transformar una página estática en una experiencia interactiva y personalizada para el usuario.
