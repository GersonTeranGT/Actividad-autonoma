Modelo de Objetos de Documento (DOM)

¿Qué es el DOM?

El DOM (Document Object Model) es una representación estructurada de un documento HTML o XML. Permite a los lenguajes de programación, como JavaScript, acceder y modificar el contenido, la estructura y el estilo de una página web. El DOM organiza los elementos como nodos en forma de árbol, donde cada etiqueta HTML es un nodo que puede tener padres, hijos y hermanos.

Caracteristícas pricnipales del DOM

Propiedades del objeto document

  • document.body: Devuelve el elemento <body>.
  • document.title: Devuelve o establece el título de la página.
  • document.imagenes: Devuelve una colección de todas las imágenes.
  • document.bgColor: Permite definir el color de fondo.
