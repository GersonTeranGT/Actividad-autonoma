Modelo de Objetos de Documento (DOM)

¿Qué es el DOM?

El DOM (Document Object Model) es una representación estructurada de un documento HTML o XML. Permite a los lenguajes de programación, como JavaScript, acceder y modificar el contenido, la estructura y el estilo de una página web. El DOM organiza los elementos como nodos en forma de árbol, donde cada etiqueta HTML es un nodo que puede tener padres, hijos y hermanos.

Caracteristícas principales del DOM

Propiedades del objeto document

  • document.body: Devuelve el elemento <body>.
  • document.title: Devuelve o establece el título de la página.
  • document.imagenes: Devuelve una colección de todas las imágenes.
  • document.bgColor: Permite definir el color de fondo.

Métodos comunes

  • getElementById(id): Selecciona un elemento por su ID.
  • getElementsByTagName(tag): Devuelve una colección de elementos por etiqueta.
  • getElementsByName(name): Devuelve elementos por su atributo name.
  • write() y writeln(): Escriben contenido HTML directamente.

Navegación entre nodos

  • parentNode: Nodo padre.
  • childNodes: Lista de nodos hijos.
  • firstChild y lastChild: Primer y último hijo.
  • previousSibling y nextSibling: Hermanos anterior y siguiente.

Manipulación de contenido

  • innerHTML: Cambia el contenido HTML de un elemento.
  • setAttribute(): Cambia el valor del atributo del elemento HTML.

    Tambien se puede cambiar el atributo o el estilo de un elemento utilizando:

    • elemento.atributo = valor;
    • elemento.style.propiedad = estilo;

Manejo de eventos
